Auto Trading을 목표로 iOS 앱을 개발하려고 하고 있어.
너는 앱 개발을 성공적으로 완수하기위해 어떤 기능들이 필요하고, 추가하면 좋을지 생각해서 
진행해야 해.

만들려고 하는 앱은 보조지표를 활용해서 매수, 매도를 자동으로 진행하는 앱이야.

App 이름 : AITrading
Target Swift Version : Swift6
Swift, SwiftUI, Combine 또는 Open Source 활용가능
MVVM 패턴으로 앱개발, 추가 기능 하기 편하게 소스 작성해줘.
데이타 저장은 SwiftData를 활용해주면 좋을거 같고,
CloudKit도 활용해주면 좋겠어.
API 통신을 바로 처리하기는 힘드니, Mockup Data로 임시로 처리할 수도 있으면 좋겠어.

소스코드 작성할 때에는 폴더위치 및 파일명까지 알려주고, 주석도 세세하게 설명해주면 좋겠어.
중간에 소스가 변경되는 경우 변경된 소스가 들어가는 위쪽 소스 아래쪽 소스도 포함해서 알려주면 좋을거 같아.
Line도 알려줄 수 있다면 알려주고. MainActor, async, await도 알맞게 사용해줘.

정보들은 첨부파일에 추가된 정보들을 활용
